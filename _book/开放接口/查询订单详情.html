查询订单详情

1. 功能说明

获取订单详情信息，可作为订单状态回调失败后的补偿，保证双方订单状态一致

2. 请求地址

https://${domain}/{customer}/{orderSource}/order/orderDetail

3. 服务协议

HTTP POST

4. 参数

参数 是否必须 字段类型 参数描述
appKey 必须 String 令牌
appSecret 必须 String 数字签名
timestamp 必须 Long 时间戳
orderId 必须 String 运单id
partnerOrderNo 必须 String 流量平台/商订单号

计费规则

1. 功能说明

查询乘客所在地点的计费规则

2. 请求地址

https://${domain}/{customer}/{orderSource}/price/priceRule

3. 服务协议

HTTP POST

4. 参数

参数 是否必须 字段类型 参数描述
appKey 必须 String 令牌
appSecret 必须 String 数字签名
timestamp 必须 Long 时间戳
fromLongitude 必须 String 出发地经度
fromLatitude 必须 String 出发地纬度
toLongitude 非必须 String 目的地经度
toLatitude 非必须 String 目的地纬度
cityId 必须 Long 城市ID
productId 必须 Integer 产品ID，由美团为运力商分配

6.返回值说明

参数名称 字段类型 说明
workday JSONObject 工作日
holiday JSONObject 休息日
startPriceDTO Object 起步价
startPriceDTO.startTime Integer 起步价包含的时间 单位 : 毫秒
startPriceDTO.startDistance Integer 起步价包含的里程 单位：米
startPriceDTO.startPrice Integer 起步价金额 单位:分
distancePriceDTO Object 里程费
distancePriceDTO.period String 里程费时段，默认普通时段为00:00-24:00
distancePriceDTO.distancePrice Integer 里程费 单位:分/公里
timePriceDTO Object 时长费
timePriceDTO.period String 时长费时段，默认普通时段为00:00-24:00
timePriceDTO.timePrice Integer 时长费 单位:分/分钟
longDistancePriceDTO Object 远途费
longDistancePriceDTO.longDistancePrice Integer 远途费 单位 分/公里
longDistancePriceDTO.longDistanceThreshold Integer 超过XX公里数，收取远途费
minPrice Integer 最低消费（与起步价互斥）

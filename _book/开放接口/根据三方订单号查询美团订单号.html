根据三方订单号查询美团订单号

1. 功能说明

根据第三方订单号查询美团订单号

2. 请求地址

https://${domain}/{customer}/{orderSource}/order/queryMTOrderId

3. 服务协议

HTTP POST

4. 参数

参数 是否必须 字段类型 参数描述
appKey 必须 String 令牌
appSecret 必须 String 数字签名
timestamp 必须 Long 时间戳
partnerOrderNo 必须 String 第三方订单号

5. 返回值

{
    "code": 200,
    "success": true,
    "msg": "成功",
    "traceId": 5857893724238462783,
    "data": "{
        "orderId": "11749994481834598433013"
    }"
}

6.返回值说明

参数名称 是否必须 字段类型 说明
orderId 必须 String 美团订单ID

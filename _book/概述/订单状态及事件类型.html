订单状态及事件类型

回调通知事件类型

事件类型 事件编码 备注 是否回调
乘客提单 5 对应订单状态10，已提单(运力商调美团) 否
调度匹配司机 10 对应订单状态15，司机匹配中 是
匹配成功 15 对应订单状态20，司机匹配成功 是
匹配失败 20 对应订单状态25，司机匹配失败 是
达成交易 25 对应订单状态30，达成交易 是
未达成交易 30 对应订单状态35，未达成交易 是
订单创建成功 35 对应订单状态40，创建订单成功 是(仅连环单有此事件)
司机出发 40 对应订单状态45 是
司机到达上车点 45 对应订单状态50，等待乘客 是
乘客上车 50 对应订单状态55，送驾中 是
变更目的地 55 乘客变更目的地触发 否
到达目的地 60 对应订单状态60，司载乘到达目的地 是
司机发起账单 65 对应订单状态65，乘客待支付 是
乘客已支付 70 对应订单状态70，乘客已支付，待评价 是
乘客评价 75 对应订单状态75，乘客已支付，已评价 否
乘客取消 80 对应订单状态80，乘客取消 否
司机取消 85 对应订单状态85，司机取消 是
客服取消 90 对应订单状态90，客服取消 是
乘客未上车 95 对应订单状态95，乘客点击未上车触发/判责判定乘客未上车 是
退款成功 105 对应退款状态15，退款成功（需要运力商支付通知回调美团) 否
客服改价 110 客服改价后，美团主动推送改价通知 是
开票成功 115 乘客申请开票后，美团主动推送开票成功状态 是
开票失败 120 乘客申请开票后，美团主动推送开票失败状态 是
发票作废成功 125 乘客申请作废后，美团主动推送发票作废成功状态 是
发票作废失败 130 乘客申请作废后，美团主动推送发票作废失败状态 是

订单取消规则

1. 功能说明

获取取消订单时的取消规则

2. 请求地址

https://${domain}/{customer}/{orderSource}/order/queryCancelRule

3. 服务协议

HTTP POST

4. 参数

参数 是否必须 字段类型 参数描述
appKey 必须 String 令牌
appSecret 必须 String 数字签名
timestamp 必须 Long 时间戳

6.返回值说明

参数名称 是否必须 字段类型 说明
pageTitle 必须 String 本订单涉及的取消规则的整体描述标题
pageContentList 必须 Array 本订单涉及的取消规则的所有内容
pageContentList[].title 必须 String 取消规则标题
pageContentList[].content 必须 String 取消规则内容
